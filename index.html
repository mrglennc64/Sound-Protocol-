STIM Statement Importer

A Royalty Statement (or Payout Report) is a critical document for publishers. Our tool imports these files directly from the STIM "My Pages" portal.

Key Data Fields Imported:

Work ID/STIM ID Swedish PRO identification
ISWC International work code
Title Song/work title
Writer(s) Creator information
Usage Type Live, Radio/TV, Streaming, Mechanicals
Territory Usage territory
Share % Publisher share
Net Amount SEK payment amount

Why this matters: Publishers spend hours manually entering this data into accounting software. If SMPT can upload a STIM Excel file, match ISWC codes to the publisher's catalog, and calculate splits, it becomes essential for publishing companies.
