Technical Integration Details

Data Fields We Process

Mandatory STIM Fields:

  • STIM Work ID - Swedish PRO identifier
  • ISWC Code - International Standard Musical Work Code
  • Title - Song/work title (Swedish/English)
  • Writer(s) - Creator information
  • Publisher Share % - STIM-assigned percentage

Financial Data:

  • Gross Amount - SEK before deductions
  • STIM Fees - Administrative charges
  • Net Amount - Payment to publisher
  • Usage Type - Radio, TV, Streaming, Live
  • Territory - Sweden, EU, International
